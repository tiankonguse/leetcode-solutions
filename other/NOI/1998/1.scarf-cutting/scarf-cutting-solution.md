# 围巾裁剪 题解

## 问题分析

从一个边长为 $N$ 的正三角形中裁出两块不重叠的正三角形小围巾，使面积之和最大。

关键约束：
- 两块小围巾都必须是正向正三角形（与原围巾方向相同）
- 不含损坏单元
- 两块不重叠

## 坐标系转换

第 $r$ 行有 $r$ 个 $\Delta$ 单元和 $r-1$ 个 $\nabla$ 单元：
- $\Delta$ 单元 $(r, j)$：第 $r$ 行第 $j$ 个正向三角，原坐标 $(r, 2j-1)$
- $\nabla$ 单元 $(r, j)$：第 $r$ 行第 $j$ 个倒向三角，原坐标 $(r, 2j)$

顶点在 $\Delta(r, j)$、边长为 $k$ 的子三角形，在第 $r+i$ 行（$i$ 从 $0$ 到 $k-1$）包含：
- $\Delta$ 单元：第 $j$ 到 $j+i$ 个（共 $i+1$ 个）
- $\nabla$ 单元：第 $j$ 到 $j+i-1$ 个（共 $i$ 个）

## 预处理最大无损边长

定义 $\text{maxK}[r][j]$：以第 $r$ 行第 $j$ 个 $\Delta$ 为顶点能裁出的最大无损正三角形边长。

利用行内前缀和逐步扩展边长：

```
maxK[r][j] = 1（若顶点完好）
对 k=2,3,...：检查底行 r+k-1 新增的 Δ 列 [j, j+k-1] 和 ▽ 列 [j, j+k-2]
若无损坏则 maxK[r][j] = k，否则停止
```

时间复杂度 $O(N^3)$。

## 核心算法：三方向分割

由**分离轴定理**：两个不重叠的凸多边形，必然可被平行于某条边的直线分开。

正三角形有三条边方向，对应三种分割方式：

### 方向1：水平线分割

枚举分割行 $\text{cut}$（$0 \le \text{cut} \le N$）：
- 上方：底行 $\le \text{cut}$ 的最大三角形面积
- 下方：顶行 $\ge \text{cut}+1$ 的最大三角形面积

### 方向2：竖直线分割（左斜边方向）

枚举分割列 $\text{cut}$（$0 \le \text{cut} \le N$）：
- 左侧：最右 $\Delta$ 列（$j+k-1$）$\le \text{cut}$ 的最大面积
- 右侧：最左 $\Delta$ 列（$j$）$\ge \text{cut}+1$ 的最大面积

### 方向3：右斜线分割（右斜边方向）

定义右斜坐标 $d = j - r$（$\Delta$ 列号减行号）。

顶点 $(r, j)$ 边长 $k$ 的三角形：
- 右边界斜坐标（最大）：$d_{\max} = j - r$（恒等于顶点处的值）
- 左下角斜坐标（最小）：$d_{\min} = j - r - k + 1$

枚举分割值 $\text{cut}$（$-N \le \text{cut} \le N-1$）：
- 左侧（$d_{\max} \le \text{cut}$）的最大面积
- 右侧（$d_{\min} \ge \text{cut}+1$）的最大面积

## 复杂度

- 时间：$O(N^3)$（预处理 maxK 和三方向前缀最大值各 $O(N^3)$）
- 空间：$O(N^2)$

$N=100$ 时总操作约 $3 \times 10^6$，远小于时限。

## 样例验证

$N=5$，损坏单元转换为 $\Delta/\nabla$ 坐标：

计算 $\text{maxK}$：
- $\text{maxK}[3][3] = 3$（顶点 $(3,5)$，边长 $3$，面积 $9$）
- $\text{maxK}[1][1] = 2$（顶点 $(1,1)$，边长 $2$，面积 $4$）

竖直分割 $\text{cut}=2$：左侧最大面积 $4$（(1,1)边长2），右侧最大面积 $9$（(3,3)边长3），总和 $13$。✓

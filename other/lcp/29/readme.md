# LCP 29. 乐团站位

某乐团的演出场地可视作 num * num 的二维矩阵 grid（左上角坐标为 `[0,0])`，每个位置站有一位成员。乐团共有 9 种乐器，乐器编号为 1~9，每位成员持有 1 个乐器。

为保证声乐混合效果，成员站位规则为：自 grid 左上角开始顺时针螺旋形向内循环以 `1，2，...，9` 循环重复排列。

请返回位于场地坐标 `[Xpos,Ypos]` 的成员所持乐器编号。
